## 9. OCR / 订阅联调专项

### 阿里云 OCR

1. `.env` 配置 `ALIYUN_ACCESS_KEY_ID/SECRET`
2. `OCR_PROVIDER=aliyun`
3. 重启 `pnpm dev:server`
4. 进件上传身份证 → 后台 OCR 字段 `provider: aliyun`

### 微信订阅消息

1. 微信公众平台申请「进件审核」「放款通知」类模板
2. `.env` 写入 `WECHAT_TEMPLATE_INTAKE_AUDIT` / `DISBURSE`
3. 配置 `WECHAT_SECRET`
4. 小程序 `MANUAL_ENV=development`，进件首页触发 `requestIntakeSubscribe`
5. 后台集成联调页显示 Token 获取成功
